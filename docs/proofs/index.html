Proof-of-Training Formal Proofs

Abstract: This collection provides rigorous mathematical foundations for the Proof-of-Training (PoT) neural network verification framework. We establish theoretical guarantees for model identity verification through challenge-response protocols, including coverage-separation theorems and wrapper detection analysis.

Core Theoretical Results

  • PDF Coverage-Separation Theorem
    Proves that challenge vectors can both cover network behavior space and separate distinct neural networks with high probability. Key result: O((diam(Θ)/ε)^p) challenges suffice for ε-verification.
  • PDF Wrapper Detection Analysis
    Analyzes security against wrapper attacks where adversaries attempt to substitute models using input/output transformations. Shows detection probability ≥ 1 - exp(-n·d²/2C) for n-dimensional challenges.

Key Theoretical Contributions

  1. Coverage Guarantee: Finite challenge sets can ε-approximate infinite network behaviors
  2. Separation Property: Distinct networks are distinguishable with probability ≥ 1 - 2^(-k)
  3. Wrapper Hardness: Perfect model wrapping is NP-hard
  4. Adaptive Detection: O(k log n) queries suffice to detect k-complexity wrappers

Implementation Notes

These theoretical results are implemented in the PoT framework at github.com/rohanvinaik/PoT_Experiments. The proofs establish security guarantees with concrete parameters suitable for practical deployment.
